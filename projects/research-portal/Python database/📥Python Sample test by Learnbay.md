# Notes 
### Beginner Level:

1. **What symbol is used to begin a comment in Python?**
   - a) `//`
   - b) `#`
   - c) `/*`
   - d) `--`
   - **Answer:** b) `#`

2. **What is the output of `print(3 * 'Hello')` in Python?**
   - a) `HelloHelloHello`
   - b) `3Hello`
   - c) `Hello`
   - d) `Hello 3 times`
   - **Answer:** a) `HelloHelloHello`

3. **Which of the following is a valid Python variable name?**
   - a) `123variable`
   - b) `_variable`
   - c) `variable-123`
   - d) `var iable`
   - **Answer:** b) `_variable`

4. **What does the `len()` function do in Python?**
   - a) Returns the length of a list
   - b) Returns the last element of a tuple
   - c) Returns the maximum value in a dictionary
   - d) Returns the number of characters in a string
   - **Answer:** a) Returns the length of a list

5. **What is the output of `2 ** 3` in Python?**
   - a) `6`
   - b) `8`
   - c) `23`
   - d) `2`
   - **Answer:** b) `8`

### Intermediate Level:

1. **Which statement is used to exit from a loop in Python?**
   - a) `exit`
   - b) `terminate`
   - c) `break`
   - d) `stop`
   - **Answer:** c) `break`

2. **What is the purpose of the `enumerate()` function in Python?**
   - a) To count the total number of elements in a list
   - b) To add indices to elements in an iterable
   - c) To merge two lists into one
   - d) To remove duplicates from a list
   - **Answer:** b) To add indices to elements in an iterable

3. **What is the output of `type(type(int))` in Python?**
   - a) `int`
   - b) `type`
   - c) `<class 'type'>`
   - d) `Error`
   - **Answer:** c) `<class 'type'>`

4. **Which data structure in Python is ordered and mutable?**
   - a) `List`
   - b) `Tuple`
   - c) `Set`
   - d) `Dictionary`
   - **Answer:** a) `List`

5. **How do you access the last element of a list named `my_list` in Python?**
   - a) `my_list[-1]`
   - b) `my_list.last()`
   - c) `my_list.last`
   - d) `my_list[len(my_list) - 1]`
   - **Answer:** a) `my_list[-1]`

### Advanced Level:

1. **What does the `*args` and `**kwargs` in a function definition refer to in Python?**
   - a) Required and optional arguments, respectively
   - b) Handling exceptions in functions
   - c) Variable arguments and keyword arguments, respectively
   - d) Decorators in Python
   - **Answer:** c) Variable arguments and keyword arguments, respectively

2. **What is the purpose of the `__init__()` method in Python classes?**
   - a) To initialize the object
   - b) To define the class
   - c) To initialize the superclass
   - d) To initialize the instance variables
   - **Answer:** a) To initialize the object

3. **Which module in Python is used for working with regular expressions?**
   - a) `regex`
   - b) `re`
   - c) `regexpy`
   - d) `regexp`
   - **Answer:** b) `re`

4. **What is the difference between `append()` and `extend()` methods in Python lists?**
   - a) `append()` adds a single element, `extend()` adds multiple elements
   - b) `append()` adds elements at the beginning, `extend()` adds at the end
   - c) `append()` and `extend()` are identical methods
   - d) `append()` adds elements in the middle, `extend()` adds at the beginning
   - **Answer:** a) `append()` adds a single element, `extend()` adds multiple elements
